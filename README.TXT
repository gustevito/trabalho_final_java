Nosso projeto é uma biblioteca com várias opções para que se tenha controle total da quantidade de exemplares e leitores, tendo todos os dados registrados e podendo verificar quantos livros há de cada tipo. É possível incluir quantos livros quiser e quantos leitores desejar. Além disso, nossa biblioteca tem um diferencial: a opção de língua inglesa, permitindo que tanto usuários que falam inglês quanto português possam acessar o nosso programa. Também possui a função "help" para ajudar e explicar como funciona para o usuário novo ou até mesmo para o usuário que esqueceu como utilizar o programa.

Dificuldades e como as superamos:

A construção desse aplicativo foi relativamente simples. Um dos nossos principais problemas era como compartilhar o código para que todos pudessem trabalhar nele. Optamos por usar o Git, já que é uma plataforma fácil de usar e a mais conhecida para esse tipo de trabalho. Além disso, enfrentamos dificuldades com erros de escrita no código ou ao chamar um método ou usar um objeto incorretamente. Para corrigir esses erros, realizamos chamadas no Discord para revisar o que estava errado e, posteriormente, corrigir. Outra dificuldade foi traduzir algumas palavras para o inglês, para a qual usamos o Google Tradutor. Tivemos também problemas ao utilizar a classe “java.util.Random”. Para isso, abrimos um site e começamos a ler como utilizá-la para criar matrículas com números aleatórios. Além disso, enfrentamos dificuldades para limpar o terminal após a execução de um comando. Para resolver isso, pesquisamos no próprio FAQ do BlueJ sobre o comando usado para limpar o terminal. Também tivemos dificuldades em criar uma interface. Para isso, utilizamos um site para escrever em ASCII e outro para obter uma arte de uma mesa com livros. Usamos o ChatGPT apenas para formatar o código para o BlueJ.

Lições aprendidas com esse trabalho:

Aprendemos a usar a classe java.util.Random para criar números aleatórios. Aprendemos também comandos para limpar o terminal após selecionar um comando. Além disso, aprendemos como escrever em ASCII, de modo a deixar uma interface mais bonita para o usuário.
